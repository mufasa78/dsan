About DSAN Technology

Dual Stream Attention Network

The Dual Stream Attention Network (DSAN) is designed for robust facial expression recognition in challenging real-world conditions.

This state-of-the-art architecture addresses common challenges in facial expression recognition such as facial occlusions (e.g., masks, glasses, hands), varying head poses, and different lighting conditions.

Key Components:
GFE-AN
Global Feature Element-based Attention

Applies sparse attention to selectively emphasize informative feature elements while suppressing those unrelated to facial expression.

MFF-AN
Multi-Feature Fusion-based Attention

Extracts rich semantic information from different representation sub-spaces to make the network insensitive to occlusion and pose variation.

Key Advantages
  • Robust to partial face occlusions
  • Handles head pose variations
  • Works in varying lighting conditions
  • Better feature extraction through attention mechanisms
  • Higher accuracy on challenging datasets
