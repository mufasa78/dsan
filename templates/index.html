About DSAN

The Dual Stream Attention Network (DSAN) is designed for robust facial expression recognition in the wild. It consists of two main components:

  • Global Feature Element-based Attention Network (GFE-AN): Applies sparse attention to selectively emphasize informative feature elements.
  • Multi-Feature Fusion-based Attention Network (MFF-AN): Extracts rich semantic information from different representation sub-spaces.

This architecture helps address challenges like facial occlusions and head pose variations that are common in real-world images.
