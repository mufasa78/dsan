About This Project

The Dual Stream Attention Network (DSAN) is designed for robust facial expression recognition in challenging real-world conditions. It addresses common problems like facial occlusions, pose variations, and lighting changes through its dual-stream architecture.

Key Components:

GFE-AN

Global Feature Element-based Attention Network selectively emphasizes informative feature elements while suppressing those unrelated to facial expression.

MFF-AN

Multi-Feature Fusion-based Attention Network extracts rich semantic information from different representation sub-spaces.
